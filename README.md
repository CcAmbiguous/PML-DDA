The code of paper "Partial Multi-Label Learning via Dual Decomposition Alignment"

先在MATLAB中运行demo.m，结果会存到deep_classifier/data文件夹里，再用Python编译类软件（如VS Code）打开deep_classifier文件夹，运行train.py。

First, run demo.m in MATLAB. The results will be saved in the deep_classifier/data folder. 
Then, use Python to compile the software (such as VS Code) and open the deep_classifier folder. Run train.py.
